# TAK-ICONS

Community-maintained ATAK/WinTAK iconsets for tactical and emergency operations.

## NWCG GeoOps Wildfire Iconset

Updated NWCG (National Wildfire Coordinating Group) GeoOps symbology for ATAK, aligned with the 2025 NIFS standards.

### Current Status

- **Latest Version**: R2 (August 2025)
- **ATAK Compatibility**: Tested on ATAK 5.4.0.16
- **WinTAK Compatibility**: Tested
- **Standards**: NWCG PMS 936 (2025)

### Key Improvements Over 2021 Version

#### New Icons Added
- **Accountable Property**: Other Property, Pump, Water Tank
- **Air Operations**: UAS Launch & Recovery
- **Water**: Draft Site, Hydrant
- **Structure Triage**: Defensible Prep & Hold, Defensible Standalone, Non-Defensible Prep & Leave, Non-Defensible Rescue Drive-by, Unknown (with labels for color-challenged users)
- **Point Repair Status**: Complete set of 7 status symbols
- **IR Points**: IR Isolated Heat Source, Possible IR Heat Source
- **Hazards**: Aerial Hazard, Hazard Tree (with corrections)

#### Icon Improvements
- Color corrections for Hazard and Hazard Tree icons (removed white background)
- Updated Aerial Hazard (yellow infill with red dot to match other hazards)
- Added text labels to Structure Triage icons for better accessibility

#### Organizational Changes
- Improved grouping/categorization for faster icon selection
- Reduced from 13 potential groups to a more manageable subset
- Logical grouping based on operational use rather than strict PMS 936 categories

### Installation

1. Download the latest iconset ZIP file from the [Releases](../../releases) page
2. Copy the ZIP file to your ATAK device
3. In ATAK, go to Settings → Import Manager
4. Select the iconset ZIP file
5. Icons will appear in your icon palette
